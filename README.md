# About Me Card - Unit 1 Project

Your Unit 1 JavaScript project! You'll build a personal "About Me" card that uses everything you learned in Unit 1.

## Getting Started

1. **Clone this repo** to your computer (or your code editor)
2. Open the folder in your editor
3. Open `about-me.html` in a browser to see what you're starting with

## Your Job

### 1. Open `about-me.html`
The structure is there, but you need to **add `id` attributes** to the elements inside the card so your JavaScript can find them. You get to pick the id names!

### 2. Don't touch `styles.css`
It's already done for you. Just let it do its thing.

### 3. Open `script.js`
This is where you'll do most of your work. The file has numbered steps with hints in the comments. Write your code under each hint.

## What You'll Use

Everything from Unit 1:
- `let` and `const`
- Math operators
- String concatenation with `+`
- `document.getElementById().textContent`
- `console.log()`
- Comments
- Bonus: `typeof`

## Stuck?

1. Open the browser console (F12) and look for errors
2. Check that the ids in your HTML match the ids in your JS **exactly** (case matters!)
3. Make sure you saved BOTH files
4. Go back to your codealong file from class and see how we did it

## When You're Done

Open `about-me.html` in a browser. You should see:
- A greeting with your name
- Your age in months
- Something about a hobby or interest
- Bonus: a small italic line showing the data type

You got this! 💪
